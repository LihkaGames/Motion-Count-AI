## 🧠 How Motion Count AI Works

Motion Count AI uses intelligent frame comparison and pixel-change analysis to detect movement inside a fixed camera frame.

### 📖 Best Usage Guide

| Step | Instructions |
|:---|:---|
| 📍 **Stable Placement** | Place your phone on a tripod, stand, or stable surface |
| 🎥 **Frame Setup** | Point the camera toward the movement area |
| ⚙️ **Calibration** | Wait for "Sensors Optimized" before starting |
| 🎚️ **Adjust Threshold** | Increase sensitivity for subtle movement or decrease false counts |
